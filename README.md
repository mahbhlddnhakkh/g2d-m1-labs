# g2d-m1-labs

## vscode

0. For nix
```sh
direnv allow
```
1. Launch the jupyter server
```sh
sh run-jupyter.sh
```

2. Copy the server URL `http://localhost:8888/lab?token=<hash>`

3. Select the kernel -> `Existing Jupyter Server` -> Put the URL there -> Select `Python 3 (ipykernel)`

Note: it will use relative paths relative to the project root.
